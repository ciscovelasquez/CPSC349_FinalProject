My Projects

Color Creator Application

An iOS application that allows a user to create a color. Through my app, you are able to modify the significance of each color (green/red/blue) through a slider or by typing it directly. The color at the top of the screen will change depending on the combination.

Technologies: Swift, Storyboard, Xcode

Challenges: Setting up the constraints for different objects

Outcome: Better able to understand the application creation process.

SmartUp

An application that helps students study better and get better grades. Similar to Quizlet, my team and I were able to create an application that allows students to add a flashcard set from which they can create a Quiz, study flashcards, matching mode, and a free response mode. They are also able to add important dates to a task manager.

Technologies: Python, GitHub, TKinter

Challenges: Being able to connect our project together as a group.

Outcome: Much better understanding of teamwork and time management.

To-List Application

An iOS application that allows you to track your daily tasks. Through this app a user is able to add tasks that they would like, checkmark tasks that have already been completed, and also delete tasks when they want to start a new task board.

Technologies: Swift, Storyboard, Xcode

Challenges: Not getting each others function to crash one another.

Outcome: Learned how to better split tasks among a group.
